⌨️ User Guide

  • Voice Input: Starts transcription at a new line at the end of the file.
  • Save (Ctrl+S): Opens the "Save As" window and tracks in history.
  • Timestamps: When ON, press Enter to tag the current recording time.
  • Focus: The cursor automatically returns to your last position after clicking any tool.
  • Language 1: Select language for Speech-to-Text transcription.
  • Convert: Translates entire editor content into Target Language.
  • Speak: Reads text aloud. If you convert, output voice switches to that language.
  • Note: If using a new language first time, voice output may take some time.
  • Shortcuts: Use Ctrl+B/I/U for Bold, Italic, and Underline.
  • Strikethrough: Press Alt+h+4 (Win) or Cmd+Shift+X (Mac).
